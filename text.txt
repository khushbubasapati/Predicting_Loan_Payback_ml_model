STEP 1 — Stop & remove everything local first
  docker compose down
  docker system prune -f
  docker image prune -a (Optional but recommended)

STEP 2 — Update docker-compose.yml
   Since images are now on Docker Hub, you should NOT use build: anymore.

STEP 3 — Pull fresh images from Docker Hub
   docker compose pull

STEP 4 — Start containers
   docker compose up -d

STEP 5 — Verify containers
   docker ps

STEP 6 — Test API
   http://localhost:8000/docs

STEP 7 — Test Streamlit UI
   http://localhost:8501


If you’re DONE testing,stop containers to free RAM + ports:
    docker compose down
